# Agent Skill Plugins

A collection of Agent Skill plugins for Claude Code and Codex.

| Plugin | Description |
|--------|-------------|
| [obsidian-kb](obsidian-kb/) | Knowledge base management via Obsidian vault — ingest sources, distill conversation insights, refine articles, check vault health |
| [lean4-skills-lite](lean4-skills-lite/) | Lightweight Lean 4 skills — library search, proof assistance |
